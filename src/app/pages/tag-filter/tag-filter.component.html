Tag Filter

A tag filter is similar to a multi-select filter but is used if there are a very large number of possible values which cannot be displayed in a multi-filter's list of options. Eg: Social Security Numbers. A column with the type of string or url can contain this Tag Filter. To add this filter to a column it is essential to add filterType as 'tag'.
