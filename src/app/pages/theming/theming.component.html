Theming

The data grid is a first class material design oriented component, therefore it ships with a light and dark theme which can be customized further to fit your companies branding and colors.

Since this grid is made with Angular Material it natively adopts to the theming configuration you provide within your application. If a new application is generated with the Angular CLI make sure to put custom theme, then you will be able to change the primary and accent color of your application and observe the grid adopt to these changes.
