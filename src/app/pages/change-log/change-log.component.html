Change Log

0.5.6, 0.5.7 (7th January 2022)

Type Description
Feature Added FixedSizeVirtualScrollStrategy for faster runtime performance while scrolling.
Feature Improved the image-url type by reducing the size of image to 40px by 40px and showing larger preview when hovered over. Refer to the documentation here.

0.5.4, 0.5.5 (12th December 2021)

Type Description
Feature Upgraded base project to Angular version 13.1.0.
Bug Fix Added z-index of 1 to mat-progress-bar because loading was not shown in some applications.
Bug Fix Fixed issue with the table header and the body not being synced on horizontal scroll on MacOS Chrome.

0.5.2, 0.5.3 (8th December 2021)

Type Description
Feature Comma separated strings in data can now be searched individually in the multi-filter and tag filters.
Feature Numeric filter may now contain numbers with upto two decimal places and negative numbers.

0.5.0, 0.5.1 (7th December 2021)

Type Description
Feature Upgraded base project to Angular version 13.0.3, The library also now compiles with Angular sources in Ivy partial compilation mode.
Feature Supported strict typing

0.4.13, 0.4.14, 0.4.15, 0.4.16 (4th December 2021)

Type Description
Bug Fix Fixed issue filtering client side paginated data when null values are present.
Feature Added ability to search empty strings in the multi-filter's filtration.

0.4.10, 0.4.11, 0.4.12 (1st December 2021)

Type Description
Bug Fix Fixed issue with checking all records checkbox automatically when filters are applied and records shown are less than the perPage count.
Bug Fix Fixed issue with filter not applying when Numeric filter equal to zero (0) is applied.

0.4.6, 0.4.7, 0.4.8, 0.4.9 (25th November 2021)

Type Description
Bug Fix Fixed the width of the pagination button by increasing the size when 3 or higher digit pages appear.
Bug Fix Fixed 'No Records Available' collapsing into 2 lines when using in a project with Bootstrap
Feature When all records in a page are selected the select all records checkbox gets automatically checked.
Bug Fix Prevent letter e from being typed in the numeric filter.

0.4.2, 0.4.3, 0.4.4, 0.4.5 (11th November 2021)

Type Description
Refactor (Breaking Changes) Removed the data type 'url' as this did not serve any purpose for an actual data type, but only helps to create links. Now urls can be generated with 'string' | 'number' | 'currency' | 'date' types. Also the clickable attribute is changed from a boolean to either the have the value 'url' or 'button'. Refer URL Builder for the updated Docs to help refactor the code.
Feature 'other' object is passed with the grid response if available.
Bug Fix Prevented the default behavior in the multi-select filter of selecting an option when the user filters the options with space being added.
Bug Fix Fixed issue with showing the value and not showing the display text when multiple multi-select filters are applied.

0.3.9, 0.4.0, 0.4.1 (9th November 2021)

Type Description
Refactor (Breaking Changes) Changed the output headingsChangedEmit to columnPreferencesChangedEmit and resetColumnPreferencesEmit to columnPreferencesResetEmit.
Feature Added new output columnPreferencesChangeEndedEmit. See how it differs from columnPreferencesChangedEmit here.
Feature The prefix and suffix attributes are now capable of showing html elements. Use this with caution to not hinder performance. For example show an image of a country in front of the country name in the column or show a red color circle when displaying 'Failed'.
Feature Added two new outputs: requestBodyEmit and sortChangedEmit. See the functionality here.

0.3.8 (7th November 2021)

Type Description
Feature Number/ currency type can now also have Tag filters. Find out here.
Bug Fix Fixed an issue of removing an incorrect tag when the user presses remove in the Tag filter.

0.3.6, 0.3.7 (5th November 2021)

Type Description
Feature Added prefix and suffix that can be added to grid data. This helps to add characters in front or at the end of data (Eg: #1 or 10%). Also the 'currency' type data can have a prefix other than $ symbol.
Feature Added an optional input to the grid component which will give the grid an elevation level shown by the intensity of the shadow underneath from 0 to 24.
Feature Added a feature to make values of the types 'string' | 'number' | 'currency' | 'date' clickable by adding a new boolean attribute named clickable to the GridHeading interface.
Bug Fix Fixed a bug relating to tag and multi-select filter types which shows as filters were applied before pressing 'filter' when search criteria is added.

0.3.4, 0.3.5 (3rd November 2021)

Type Description
Feature Added button to reset column preferences which will trigger an output that can be captured from parent component when clicked.
Feature Added feature to reinitialize grid by resetting all added filters from parent component.

0.3.3 (30th October 2021)

Type Description
Bug Fix Fixed the issue of not attaching a token (if available) when calling an endpoint from the string multi-select filter.
Feature Added currency data type.
Feature Added headerTooltip attribute to GridHeading interface to pass a small explanation for the column header which will be shown when the user hovers over.
Refactor Increased the width of column control menu.

0.3.2 (16th September 2021)

Type Description
Feature Added row reserved keyword amdgGridRowBackground to highlight an entire row with a given background color. Check Cell Styles example.

0.3.1 (14th September 2021)

Type Description
Refactor (Breaking Changes) Changed align attribute to textAlign within the GridHeading interface.
Feature Added background, color, fontWeight, fontStyle attributes to GridHeading interface to change the styles of an entire column. Check Cell Styles example.
