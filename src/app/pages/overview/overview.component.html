Advanced Angular Material

Data Grid System (v0.3.1)

Inspired by Kendo UI Grid

Built-in Virtual Scrolling

Scroll through large numbers of rows smoothly without having a hit on the page performance. You don't have to configure anything extra.

Advanced Multi-Filter Features

This grid system ships with advanced string filters, numeric filters, date range filters, multi-selection filters and tag filters which are extremely easy to configure. The documentation is also very detailed and helpful.

Customizable Theming

The data grid is a first class material design oriented component, therefore it ships with a light and dark theme which can be customized further to fit your company's branding and colors.

Optional Columns with Re-ordering

When data grids have a large number of columns, some can be made optional so the users would not get overwhelmed. These can be shown if the user wishes and the user can also customize the view by re-ordering the columns.

Master Detail Grid

The grid is capable of making each row expandable to show more details about a particular record. The developer has the ability to make the expanded row contain a custom HTML template with the option of adding row variables (using syntax similar to Angular's String interpolation) or open a child grid with most of the capabilities of the original grid.
